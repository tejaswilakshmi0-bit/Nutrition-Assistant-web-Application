# Nutrition Assistant Web Application

## Project Overview

Nutrition Assistant is a full-stack MERN web application that helps users monitor their nutrition intake, manage meals, and receive AI-powered diet recommendations. The application provides secure authentication, meal tracking, and personalized nutrition insights.

## Features

* User Registration and Login
* JWT-Based Authentication
* Meal Tracking and Management
* Nutrition Analysis
* AI-Powered Diet Plan Generator
* Responsive User Interface

## Technologies Used

### Frontend

* React.js
* Vite
* CSS

### Backend

* Node.js
* Express.js

### Database

* MongoDB

### Authentication

* JWT (JSON Web Token)

## Installation

### Clone Repository

git clone <repository-url>

### Backend Setup

npm install
npm run dev

### Frontend Setup

npm install
npm run dev

* Frontend (React + Vite)
